# May I? app privacy policy

May I? uses Firebase authentication for user login. The preferred sign in is Google Play Games or Apple Game Center. When signing in with those, the app developers have no identifiable information. When signing in with email and password, they are secured by Firebase and the developers can see emails. To delete an email from the record, request so [here](https://docs.google.com/forms/d/e/1FAIpQLSd0Z_7ufzsrUblGiWXhoH42tGIyBktCNJ_284R8vahYT5ltnw/viewform?usp=dialog).

All data collected for may I is related to the game play and nothing else. It is tied to an anonymous user ID.

May I? uses Google Crashlytics to help me fix bugs, the data captured is anonymous.
